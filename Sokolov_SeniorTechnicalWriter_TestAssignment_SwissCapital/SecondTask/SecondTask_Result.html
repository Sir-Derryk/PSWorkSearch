Add a user

Split 'Add' and 'Edit' into separate sections for better structure.
Headings

  1. Navigate to Users > New user.
  2. Specify the user's email address.
  3. Specify a temporary password. The user can change this password later.
  4. Complete the profile fields:
    • First name: 1-128 characters. Letters, apostrophes, hyphens, periods, and spaces only.
    • Last name: 1-128 characters. Letters, apostrophes, hyphens, periods, and spaces only.
    • Date of birth: Select from the calendar.
    • Country: Specify the user's country.
    • City: Specify the user's city.
    • Address: Specify the user's address.
    • Phone number: Specify the user's phone number.
    • Gender: Select Male or Female.
    • Additional information: If needed, specify additional information, such as nickname, profession, time zone, or security question.
  5. If needed, change the default Personal ID assigned to the user.
  6. Click Create to apply your changes.

Used 'select' for choose in sub-form, 'specify' for typing, 'click' for buttons. Used "First name" and 'Last name' instead of "user name" and "surname" (this should be done for the UI as well).

Edit a user

  1. Navigate to Users.
  2. Select the email address of the user you want to edit.
  3. On the user details page, click Edit user.
  4. Make your changes to any of these fields:
    • First name, last name, nickname
    • Date of birth
    • Country, city, address, phone number
    • Gender, profession, time zone
    • Security question, personal ID
  5. Click Update user to apply your changes.

Made editing a distinct process with its own heading. Restructured steps. Grouped related fields for easier scanning. Used 'click' for buttons.
